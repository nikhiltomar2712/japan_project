# Japan Project

## Overview
The Japan Project is an application designed to provide information about various places in Japan. It serves as a platform for users to explore and learn about the cultural and historical significance of different locations.

## Features
- Explore various places in Japan
- Detailed information about each location
- User-friendly interface

## Installation
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/nikhiltomar2712/japan-project.git
   ```
2. Navigate to the project directory:
   ```
   cd japan-project
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```
python src/main.py
```
